For Python development and testing you can and should use this Virtual environment (venv).<br/>
1. First you need to download "pip" (recommendation: pip3) and after that install virtualenv by command "pip3 install virtualenv" (change "pip3" to "pip" if not working) to your computer.  <br/>
2. Second you need to activate venv. To activate venv you need to navigate to "snowledge" folder. Then you need activate venv by command "source venv/bin/activate". <br/>
(3. If you need to download more tools or modules you can download them after succesfull activate by pip.)
<br/>
4. To download all necessary modules you need to run command "pip install -r requirements.txt"
<br/>
(5. If you downloaded more modules or tools you need to download them to requirements.txt order to help other developers of this project. Order to do that you should run command "pip freeze > requirements.txt").
<br/>
6. Last step is to set correct python interpreter. This step depends on which editor or IDE you are using, but you can find help from internet pretty easily. (Correct path to interpreter snowledge/venv/bin/"python3.10" or any other python which you want to use. I recommended to use python 3.10 because there are no conflicts with requirements.txt versions.)
<br/>
After that you should have every modules and tools which are necessary with python on this project!
<br/>
RUN BACKEND USING "python3".
